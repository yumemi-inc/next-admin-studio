## 管理項目

| 項目 | 説明 | 入力の種類 |
| ---- | ---- | ---- |
| 管理名称 | 管理上の名称（一覧画面で表示される） | テキスト入力 |
| 公認アーティスト | 公認されたアーティストかどうかのフラグ | スイッチ |
| アイコンURL | アイコン画像のURL | テキスト入力 |
| 名前 | アーティストの名前 | テキスト入力 |
| タグ | アーティストに紐づくタグ | タグ入力 |
